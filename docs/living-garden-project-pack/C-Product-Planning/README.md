# Living Garden Product Planning

This folder defines what the customer experiences from Day 1 to Day 35.

- It does not define UI design.
- It does not contain image-generation prompts.
- It does not contain Claude Code implementation instructions.

The core customer experience is simple:

> The customer enters Living Garden and notices that the garden has become slightly richer than before.

- No complicated story is required.
- No long explanation is required.
- No achievement popup is required.

The visual change itself is the reward.
