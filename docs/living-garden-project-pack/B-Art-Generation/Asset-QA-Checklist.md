# Living Garden Asset QA Checklist

For each asset, mark **Pass** or **Revise**.

## Visual Consistency

- [ ] Matches the approved Living Garden reference
- [ ] Looks like part of the same storybook world
- [ ] Does not resemble unrelated clip art
- [ ] Does not resemble a game inventory icon
- [ ] Does not look 3D or plastic

## Color

- [ ] Medium-rich saturation
- [ ] Not pale or washed out
- [ ] Not neon or excessively saturated
- [ ] Uses the approved natural palette
- [ ] Has enough contrast against garden greenery

## Lighting

- [ ] Warm golden morning light
- [ ] Upper-left lighting direction
- [ ] Soft natural shadow
- [ ] No harsh black shadow
- [ ] No cold or dramatic lighting

## Shape

- [ ] Clear silhouette
- [ ] Readable when reduced to mobile size
- [ ] Organic curves
- [ ] Natural asymmetry
- [ ] No rigid or artificial geometry

## Technical

- [ ] Transparent background
- [ ] No white background
- [ ] No visible white fringe
- [ ] No text
- [ ] No logo
- [ ] No watermark
- [ ] Full object visible
- [ ] Safe transparent padding
- [ ] Correct filename
- [ ] PNG format

## Final Decision

Use one of:

- **APPROVED**
- **REVISE**
- **REJECTED**

Do not integrate any asset that has not been marked **APPROVED**.
